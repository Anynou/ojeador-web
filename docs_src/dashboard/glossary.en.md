# Glossary

- **player_id**: unique player identifier in the dataset. Used for mini profile, compare, inspector, etc.
- **Scouting Index**: aggregated signal to prioritize performance/fit.
- **Upside**: potential/growth signal.
- **Context (league)**: compares player performance against league mean/expected metrics.
- **Scope (similars)**: similarity level (league, country, continent, global).
- **Saved searches**: browser-saved filter sets for reuse.
- **Activity**: local log (last actions) for quick recall.
