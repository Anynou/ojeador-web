# Jugadores (Free)

Pantalla de filtrado rápido para crear shortlists.

## Qué puedes hacer

Filtrar por:

- nombre, equipo, liga
- país, continente
- posición, pie
- bandas de edad (`U21`, `22–24`, `25–29`, `30+`)
- bandas de altura (`<170`, `170–179`, `180–189`, `190+`)
- ordenación (por nombre, etc.)

Paginar resultados (limit/offset).

Abrir **mini-ficha** de un jugador al seleccionarlo.

Guardar búsquedas (“Saved searches”) en el navegador.

Exportar resultados a CSV.

## Mini-ficha de jugador

Al seleccionar un jugador, se cargarán:

- datos básicos
- posiciones

## Guardar búsquedas (Saved)

- Se guardan en `localStorage` (máx. 20 por scope).
- Scope de esta pantalla: `jugadores`.

!!! tip "Convención recomendada para nombres"
    Usa un nombre que explique el objetivo:  
    `Central zurdo U23 · Portugal · presupuesto bajo`
