# Overview

## What you see

- Dataset **KPIs**: players, countries, competitions, teams.
- **By continent** distribution (bars + percentages).
- **Recent activity** (last actions saved locally in your browser).

## Quick actions

From the hero block you can jump to:
- Players
- Scouting Search (if allowed by plan)
- Compare (if allowed by plan)


## Notes

- “Activity” is stored in `localStorage` (max 10 items).
- If continent data is missing, maybe the dasaset is in a upgrade.

!!! tip "Practical use"
    Use Overview to quickly validate the dataset is reachable and to explain coverage (by continent) to staff/clients.
