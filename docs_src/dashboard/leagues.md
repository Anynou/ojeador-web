# Ligas (Free)

Dos modos para analizar competiciones:

## Modo Ranking
Pensado para ver un ranking compacto de ligas (con *fuerza de liga*).

- Filtros: país y liga (texto)
- Parámetros principales: `limit`

## Modo Metricas
Pensado para explorar métricas con paginación.

- Filtros: país y liga (texto)
- Parámetros principales: `limite` (máx 500 en el dashboard) y `offset`


## Consejos de uso

- Si necesitas una lista rápida “top ligas” por volumen o fuerza: **Ranking**
- Si estás haciendo análisis y quieres navegar por páginas: **Metricas**

!!! note "Cuando veas pocos resultados"
    Si el filtro de país/liga es muy estricto, prueba con texto parcial (ej. `Spain` / `Liga`).
