# Dashboard · Documentación

Esta sección documenta el dashboard de ElOjeador: qué hace cada pantalla, qué permisos necesita y cómo usarlo en un flujo real de scouting.

!!! tip "Atajo rápido"
    Si es tu primera vez, ve a **[Primeros pasos](quickstart.md)**.

## Qué incluye el dashboard

- **Resumen**: KPIs del dataset + distribución por continente + actividad reciente.
- **Jugadores (Free)**: buscador con filtros, mini-ficha y export CSV.
- **Ligas (Free)**: ranking de ligas y métricas por competición.
- **Scouting Search (Pro/Business)**: búsqueda avanzada con **Scouting Index** y **Upside**, contexto de liga y similares.
- **Comparar (Pro/Business)**: comparación 1v1 por `player_id`.
- **Player Inspector (Business)**: “inspector” que intenta cargar Free + Scouting + Private (best-effort).
- **Upgrade/Facturacion**: compra/gestión de suscripción (Stripe).

## Permisos por plan

- **Free**: Resumen, Jugadores, Ligas.
- **Pro**: + Scouting Search, Comparar.
- **Business**: + Player Inspector.

## Idioma

El dashboard incorpora selector **ES/EN** en la barra superior (si estás logueado).
