# Players (Free)

Fast filtering screen to build shortlists.

## What you can do

Filter by:

- name, team, league
- country, continent
- position, foot
- age bands (`U21`, `22–24`, `25–29`, `30+`)
- height bands (`<170`, `170–179`, `180–189`, `190+`)

- sorting (name, etc.)

Paginate results (limit/offset).

Open a player **mini profile** when selecting a row.

Save searches (“Saved searches”) locally.

Export results to CSV.

## Mini profile

When you select a player, the dashboard tries to load:

- basic info
- positions

## Saved searches

- Stored in `localStorage` (max 20 per scope).
- Scope for this screen: `players`.

!!! tip "Naming convention"
    Use goal-oriented names, e.g.  
    `Left-footed CB U23 · Portugal · low budget`
