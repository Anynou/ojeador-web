# Player Inspector (Business)

Vista tipo “inspector” orientada tener la información más detallada de un jugador.

## Qué hace

Para el `player_id` introducido intenta mostrar, en este orden:

1. **Mini resumen**
2. **Perfil Scouting**
3. **Contexto**
4. **Detalles**
5. **Contrato**
6. **Últimos partidos**
7. **Trayectoria**


!!! warning "Uso recomendado"
    Esta pantalla es muy útil para tener los detalles de un jugador.
    Para un usuario final suele ser “demasiado técnica”.
