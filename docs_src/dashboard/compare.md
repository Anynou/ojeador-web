# Comparar (Pro/Business)

Comparación 1v1 a partir de dos `player_id`.

## Cómo usarlo

Obtén los `player_id` desde:

   - la tabla de Players (Free) o Scouting Search
   - el export CSV

Pega un id en el campo A y otro en el campo B.

Ejecuta la comparación.

## Qué muestra

- Información básica de ambos jugadores
- Scouting Index / Upside
- `pillars` (bloques de rendimiento: performance, form, progression, reliability, trend)
- tags de perfil (`profile_tags`)


!!! note "Si solo devuelve 1 jugador"
    Normalmente significa que uno de los IDs no existe o no está disponible para tu plan.
