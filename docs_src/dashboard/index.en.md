# Dashboard · Documentation

This section documents the **ElOjeador web dashboard**: what each screen does, which permissions it requires, and how to use it in a real scouting workflow.

!!! tip "Quick path"
    First time here? Start with **[Getting started](quickstart.md)**.

## What the dashboard includes

- **Overview**: dataset KPIs + continent distribution + recent activity.
- **Players (Free)**: search with filters, mini profile and CSV export.
- **Leagues (Free)**: league ranking and competition metrics.
- **Scouting Search (Pro/Business)**: advanced search with **Scouting Index** and **Upside**, league context and similars.
- **Compare (Pro/Business)**: 1v1 comparison by `player_id`.
- **Player Inspector (Business)**: “inspector” that tries Free + Scouting + Private (best-effort).
- **Upgrade/Billing**: buy/manage subscription (Stripe).
- **Settings (Admin)**: configure API Base URL and API Key.

## Permissions by plan

- **Free**: Overview, Players, Leagues.
- **Pro**: + Scouting Search, Compare.
- **Business**: + Player Inspector.

## Language

The dashboard includes an **ES/EN** language toggle in the top bar (when logged in).
