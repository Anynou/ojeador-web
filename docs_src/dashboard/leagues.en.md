# Leagues (Free)

Two modes to analyze competitions.

## Rank mode
Quick compact ranking of leagues (with *league_strength*).

- Filters: country and league (text)
- Main param: `limit`


## Metrics mode
Explore league metrics with pagination.

- Filters: country and league (text)
- Main params: `limit` (max 500 in the dashboard) and `offset`

## Tips

- Need a quick “top leagues” list: use **Rank**
- Need to browse with pages: use **Metrics**

!!! note "If you see too few rows"
    Try partial text filters (e.g. `Spain` / `Liga`).
