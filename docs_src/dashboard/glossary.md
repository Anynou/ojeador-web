# Glosario

- **player_id**: identificador único del jugador en el dataset. Se usa para mini-fichas, compare, inspector, etc.
- **Scouting Index**: señal agregada para priorizar rendimiento/encaje.
- **Upside**: señal de potencial/mejora.
- **Context (liga)**: comparación del rendimiento del jugador con métricas media/esperada de su competición.
- **Scope (similares)**: nivel de comparación para “similar players” (liga, país, continente, global).
- **Saved searches**: búsquedas guardadas en el navegador para reutilizar filtros.
- **Activity**: log local (últimas acciones) para tener rastro de uso.
