# Player Inspector (Business)

A technical “inspector” view for a deeper information about one player.

## What it does

For the provided `player_id`, it tries (in order):

1. **Free mini**
2. **Scouting profile**
3. **Scouting context**
4. **Private detail**
5. **Contract**
6. **Matches**
7. **Career**


!!! warning "Recommended use"
    Great for player details in deep.
    For end users it may be too technical.
