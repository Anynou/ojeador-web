# Upgrade y facturación

El dashboard incluye una pantalla para comprar o gestionar el plan del **Dashboard**.

## Upgrade

- Si estás en **Free**: aparecen botones para **Comprar Pro** o **Comprar Business**.
- Si estás en **Pro/Business**: aparece botón para **Gestionar suscripción**.

Todas las suscripciones se gestionan a través de la plataforma Stripe

En cualquier momento se puede cambiar entre suscripciones o cancelar.


!!! note "Si el plan no cambia al instante"
    A veces Stripe/webhooks tardan unos segundos.  
    Vuelve al dashboard, refresca la página y revisa tu tier.
