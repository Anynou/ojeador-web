# Overview

## Qué ves aquí

- **KPIs** del dataset: jugadores, países, competiciones y equipos.
- **Distribución por continente** (barras con porcentaje).
- **Actividad reciente** (últimas acciones guardadas en el navegador).

## Acciones rápidas

Desde el bloque superior puedes ir a:
- Jugadores
- Scouting Search (si tu plan lo permite)
- Comparar (si tu plan lo permite)

## Notas

- La sección “Actividad” se guarda en `localStorage` (máx. 10 items).
- Si no hay datos por continente, puede ser que haya alguna actualización de datos en curso.

!!! tip "Uso práctico"
    El overview sirve para validar rápidamente que el dataset está cargado y para explicar el alcance (cobertura por continente) a un staff o cliente.
